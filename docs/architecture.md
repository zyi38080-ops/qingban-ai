# 晴伴AI 系统架构文档

```
┌─────────────────────────────────────────────┐

│  ┌───────────────────────────────────────┐  │

│  │  │ChatWindow│ │Privacy│ │Workflow   │  │  │
│  │  │    │     │ │Panel │ │Panel      │  │  │
│  │  │    │     │ └──────┘ └───────────┘  │  │
│  │  │    ▼     │                          │  │
│  │  │ 脱敏工具  │  PlanCard  KnowledgeCard │  │
│  │  │ 风险识别  │                          │  │
│  │  │ 知识检索  │                          │  │
```

## 2. 工作流

```
用户输入 → 隐私脱敏 → 风险识别 → 知识库检索 → Qwen3.7-plus 生成回答 → 行动计划生成 → 展示回复
```

### 2.1 隐私脱敏
- 使用正则表达式识别敏感信息
- 自动替换手机号、邮箱、身份证、银行卡等
- 不保存脱敏前的原始输入

### 2.2 风险识别
- 基于关键词规则（非AI分类）
- 识别自杀、自残、暴力等高风险表达
- 触发后进入安全分流，不调用LLM

### 2.3 知识库检索
- 本地 JSON 知识库（12条）
- 基于关键词匹配
- 每次返回最多2条

### 2.4 LLM 调用
- 通过 Cloudflare Worker 代理转发
- API Key 存储在 Worker 环境变量中
- 降级：无 API Key 时进入 Demo Mode

### 2.5 行动计划生成
- 基于规则模板匹配
- 用户说"帮我制定计划"时展示
- 内容简短、具体、可执行

## 3. 隐私设计

- 默认不保存聊天内容
- 默认不保存真实身份信息
- 用户主动授权后才保存偏好
- 前端脱敏，不向后端传输敏感信息
- LocalStorage 存储，用户可随时清空

## 4. 技术选型原因

| 选择 | 原因 |
|------|------|
| Qwen3.7-plus | 课程要求，阿里云百炼平台 |
| Cloudflare Worker | 免费额度，零运维，保护 API Key |
| React + Vite | 快速开发，生产级打包 |
| 本地 JSON 知识库 | 满足课程要求，无需向量数据库 |
| GitHub Pages | 免费静态部署，自动 HTTPS |
| Demo Mode | 无 API Key 时仍可演示 |
